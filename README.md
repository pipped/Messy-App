# MessyApp

MessyApp is an NFC-based wardrobe management application developed as a senior design project. It connects physical clothing items to a digital system and enhances wardrobe organization through weather-based recommendations and outfit generation.

## Overview

MessyApp bridges physical and digital systems using NFC technology. Clothing items are tagged and can be scanned to manage and track a personal wardrobe. The application also integrates external data to provide intelligent recommendations for what to wear.

## Features

- NFC scanning to identify and track clothing items  
- Digital wardrobe organization and inventory management  
- Weather integration for context-aware clothing suggestions  
- Outfit generation based on available items and conditions  
- Web-based interface for managing wardrobe data  

## Tech Stack

- Frontend: (React)  
- Backend: (Node.js, Express, etc.)  
- Database: (MongoDB, Firebase, etc.)  
- APIs: Weather API integration  
- Hardware: NFC tags / NFC-enabled device
